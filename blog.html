Blog de difusión de contenidos

Jesús García Minas - 27/04/2021

Bienvenidos a mi blog para la asignatura de Difusión de contenidos multimedia vía web. Que mejor manera que estrenar el blog explicando el propio blog que estoy desarrollando para la asignatura de Difusión de Contenidos Web del Master Universitario de Ingeniería Web que estoy cursando.

El blog está compuesto de distintos ficheros html. Uno por cada una de las pantallas de este blog. Además, también se incluyen ficheros html para las presentaciones realizadas con RevealJS. Por otro lado, se cuenta con un fichero de estilos (css) para dotar a la página web de un estilo propio.

En cuanto a la navegabilidad, la página web cuenta con un menú superior con diferentes secciones (ninguna operativa) lo que permite al usuario volver a la página de inicio desde cualquier de los artículos escritos.

En cuanto a la pantalla principal, además de un resumen de cada artículo y el enlace para acceder a estos, se cuenta con una tarjeta con información del dueño del blog además de enlaces de interés tanto de ámbito academíco (Universidad de Oviedo, Escuela de Ingeniería Informática y Máster) como de ámbito personal (GitHub y LinkedIn).

Por último, se cuenta con un footer general también con información del dueño del blog.
